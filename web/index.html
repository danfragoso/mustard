Layout
How does the mustard ui layout works?

On mustard ui there is only one type of layout, the box.

The two main concepts of the box layout is it's orientation and it's alignment.

Orientation is controlled by the componentProp with the same name, it can be either Horizontal or Vertical. This property controls how it's children will be rendered, for example this is how childs in a Horizontal layout should be layed.

The Horizontal Orientation is the default one, in this case, the Orientation componentProp can be ommited.


import . "mustard"

func Child() {
  return Box(&ComponentProps{
    Background: nameToColor("coral"),
    Height: 50, Width: 50,
    MarginTop: 10, MarginRight: 10, MarginLeft: 10, MarginBottom: 10,
  })
}

func HorizontalComponentExample() {
  return Box(&ComponentProps{
      Background: NameToColor("cornflowerblue"),
    },
    Child(),
    Child(),
  )
}

And the example below is the Vertical layout


import . "mustard"

func Child() {
  return Box(&ComponentProps{
    Background: nameToColor("coral"),
    Height: 50, Width: 50,
    MarginTop: 10, MarginRight: 10, MarginLeft: 10, MarginBottom: 10,
  })
}

func HorizontalComponentExample() {
  return Box(&ComponentProps{
    Background: NameToColor("cornflowerblue"),
      BoxProps: &BoxProps{
        Orientation: Vertical,
      },
    }
    Child(),
    Child(),
  )
}

For alignment, the values for the Alignment componentProp can be Start, Center or End.

Start Alignment

The Start Alignment is the default one, in this case, the Alignment componentProp can be ommited.


&ComponentProps {
  ...,
  BoxProps: &BoxProps{
    Alignment: Start,
               ^^^^^
  },
}

Center Alignment


&ComponentProps {
  ...,
  BoxProps: &BoxProps{
    Alignment: Center,
               ^^^^^
  },
}

End Alignment


&ComponentProps {
  ...,
  BoxProps: &BoxProps{
    Alignment: End,
               ^^^^^
  },
}
